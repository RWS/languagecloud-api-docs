    Undefined

    Represents an undefined tag type. This is also used for "dummy tags" which may be added to a segment in case a tag was deleted from the original segment. Such a situation occurs when the translation engine deletes tags, but the number of tokens must remain constant in the translation proposal as otherwise the token positions will become invalid.
